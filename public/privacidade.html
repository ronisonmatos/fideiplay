3. Dados de Menores

O SantosPlay não permite o cadastro de usuários com menos de 13 anos. Usuários entre 13 e 18 anos devem ter o conhecimento de seus responsáveis legais, conforme exigido pelo art. 14 da LGPD.

Caso identifiquemos o cadastro de um menor de 13 anos, a conta será removida imediatamente. Responsáveis podem entrar em contato pelo e-mail ronisonmaria@gmail.com para solicitar a exclusão.

4. Como os Dados São Armazenados

Os dados são armazenados de forma segura na plataforma Supabase (infraestrutura em nuvem com criptografia em trânsito via TLS e em repouso). Senhas nunca são armazenadas em texto claro — utilizamos o sistema de autenticação segura do Supabase Auth.

Alguns dados de progresso também são armazenados localmente no dispositivo via AsyncStorage para funcionamento offline.

5. Compartilhamento de Dados

Não vendemos nem comercializamos seus dados pessoais. Compartilhamos dados apenas com:

  • Supabase — banco de dados e autenticação. Atuam como operadores de dados sob contrato (Política do Supabase).
  • Redes de anúncios — quando você assiste a anúncios voluntários para ganhar moedas, a rede de anúncios pode coletar dados de forma independente, sujeita à política de privacidade própria da rede.
  • Autoridades competentes — quando exigido por lei ou ordem judicial.

7. Retenção de Dados

Seus dados são mantidos enquanto a conta estiver ativa. Após a exclusão da conta, os dados pessoais são removidos em até 30 dias, exceto quando houver obrigação legal de retenção.

8. Cookies e Armazenamento Local

O app utiliza armazenamento local no dispositivo (AsyncStorage) para salvar preferências de tema, progresso de jogos e dados de sessão. Não utilizamos cookies de rastreamento ou tecnologias de terceiros para fins de publicidade comportamental na versão mobile.

9. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, perda ou divulgação indevida, incluindo:

  • Comunicação criptografada via TLS/HTTPS
  • Autenticação gerenciada pelo Supabase Auth com hashing de senhas
  • Controle de acesso por linha (Row Level Security) no banco de dados

10. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Quando houver alterações relevantes, informaremos dentro do app. O uso continuado após a publicação implica aceitação das mudanças. A data de última atualização sempre estará indicada no topo desta página.

11. Autoridade Nacional de Proteção de Dados (ANPD)

Você tem o direito de peticionar à ANPD em caso de violação desta Política ou da LGPD.
